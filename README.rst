Dotfiles
========

My personal dotfiles in a setup with Arch Linux on an Asus X515

Quick Start
-----------

To install, clone the repository:

.. code-block:: bash

	git clone https://github.com/Henriquehnnm/dotfiles.git


And copy the configuration files to their respective places manually.
