# Henriquehnnm's Dotfiles
## Simply beautiful

These are dotifles based on [end-4/dots-hyprland](https://github.com/end-4/dots-hyprland), adapting for my daily use of studies and front-end web development.

### Installation

- To install on Arch, first install the base with the command `bash <(curl -s https://ii.clsty.link/get)`
- And apply the settings manually to each respective file.
